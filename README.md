# Student Management System

A simple and user-friendly Student Management System built using HTML, CSS, and Vanilla JavaScript. This project helps users manage student records efficiently with a clean and interactive interface.

---

## Live Demo  

https://jerosesandra18.github.io/student-management-system

---

## Features

- Add new student details  
- View all student records  
- Delete student entries  
- Data stored using Local Storage  
- Clean and aesthetic user interface  

---

## Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---

## Project Structure

student-management-system/  
│── README.md  
│── index.html  
│── style.css  
│── script.js  

---

## How to Run Locally

1. Download or clone the repository  
2. Open the project folder  
3. Open index.html in your browser  

---

## Project Objective

To build a basic student management system and practice frontend development concepts like DOM manipulation, event handling, and data storage.
